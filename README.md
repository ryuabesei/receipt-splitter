# Receipt Splitter

買った商品を手入力して、2人の立て替え精算を計算する静的アプリです。

## できること

- 商品名を入力
- 値段を入力
- 2人のもの / 自分だけ / 相手だけを選択
- 自分が払った / 相手が払ったを選択
- 最後にどちらがどちらへ何円払うべきかを自動計算
- ローカルアカウントを登録してログイン
- アカウントごとに2人の名前と履歴を保存
- 完了した精算を過去履歴として保存

## 開き方

`index.html` をブラウザで開くだけで使えます。APIキーやサーバーは不要です。

アカウント、名前、履歴はブラウザの `localStorage` に保存されます。同じ端末・同じブラウザ内でログインして切り替えられます。
